What is Mira?

Mira is a modern, open-source, self-hosted link shortener for individuals, teams, and organizations. It provides secure, easy, and powerful link management with advanced features for everyone.

  • Admin & User management, role-based permission
  • Random or custom short links
  • Password protection, expiration, one-time links
  • Personal dashboard, user dashboard, analytics
  • RESTful async API for automation
  • Google Safe Browsing for security
  • Blacklist for domains
  • Email notification for account creation & password reset
  • Beautiful pastel UI with Tailwind CSS
  • Easy PostgreSQL setup with .env configuration
