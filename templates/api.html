API Documentation

Use our API to shorten URLs programmatically. Here is an example endpoint:

POST https://filin.fyi/api/shorten
Headers:
  Content-Type: application/json
  X-API-Key: 
your_api_key

Body:
  {
    "url": "https://example.com",
    "alias": "customAlias",          // optional
    "password": "optionalPassword",  // optional
    "max_clicks": 0,                 // optional, 0 means unlimited
    "expire_at": "2025-12-31T23:59:59" // optional, ISO8601 format
  }

The API will return a JSON object containing the shortened URL.
